# AI Academy Website - Anote Template

This is a full-stack application template for building AI Academy-style platforms using React frontend and Flask backend.

## Quick Start

### Frontend Setup
```bash
cd frontend
npm install
npm start
```

### Backend Setup
```bash
cd server
pip install -r requirements.txt
flask run
```

### Documentation
```bash
cd docs
mkdocs serve
```

## Architecture
- **Frontend**: React with Tailwind CSS
- **Backend**: Flask API with MySQL
- **Documentation**: MkDocs
- **Deployment**: AWS S3, CloudFront, ECR, Elastic Beanstalk
